5

Click:

Edit system environment variables

Click:

Environment Variables

Under System Variables → New

Variable Name: JAVA_HOME
Variable Value: (paste copied path)
🔹 Step C: Add to PATH
In System Variables

Select:

Path → Edit → New

Click OK → OK → OK
