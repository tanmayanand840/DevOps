# 🧪 **Gradle Project Setup**

---

## 🔹 Step 1: Open VS Code

* Open Visual Studio Code
* Go to **Extensions**

---

## 🔹 Step 2: Install Required Extensions

Install:

* Gradle for Java
* Extension Pack for Java
* Groovy

---

## 🔹 Step 3: Create Project

* Open a **new folder**
* Click **New File / Create Java Project**
* Select:

  * Java Project
  * Gradle
  * Groovy
* Press Enter

👉 This creates your Gradle project

---

## 🔹 Step 4: Go to File Location

From your notes:

```
app → src → main → java → App.java
```

Open **App.java**

---

## 🔹 Step 5: Write the Following Code

```java
import java.awt.Desktop;
import java.net.URI;

public class App {
    public static void main(String[] args) {
        try {
            String url = "https://www.google.com";

            if (Desktop.isDesktopSupported()) {
                Desktop desktop = Desktop.getDesktop();
                desktop.browse(new URI(url));
            } else {
                System.out.println("Desktop not supported");
            }

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

---

## 🔹 Step 6: Add Gradle Task

Open `build.gradle` and add at the bottom:

```groovy
task hello {
    doLast {
        println("HelloGradle");
    }
}
```

---

## 🔹 Step 7: Run Commands

Open terminal in VS Code:

### Run task:

```bash
gradle hello
```

### Run project:

```bash
gradle run
```



# 🧪 **Manual Gradle Installation (Corrected)**

## 🔹 Step 1:

Search **“Gradle download”** in browser

---

## 🔹 Step 2:

Download the **binary-only zip file**

---

## 🔹 Step 3:

Extract the zip file and copy it into:

```text
C:\Gradle
```

Example:

```text
C:\Gradle\gradle-8.x
```

---

## 🔹 Step 4:

Open the folder → go to **bin** folder and copy the path:

```text
C:\Gradle\gradle-8.x\bin
```

---

## 🔹 Step 5: Set Environment Variables (IMPORTANT ⚠️)

1. Search **“Edit System Environment Variables”**
2. Click → **Environment Variables**

### 👉 In System Variables:

#### (A) Add GRADLE PATH

* Select **Path** → Click **Edit**
* Click **New**
* Paste:

```text
C:\Gradle\gradle-8.x\bin
```

---

#### (B) (Optional but better) Add GRADLE_HOME

* Click **New**

```text
Variable Name: GRADLE_HOME
Variable Value: C:\Gradle\gradle-8.x
```

---

### 👉 Final Step:

Click **OK → OK → OK**

---

## 🔹 Step 6: Verify

Open CMD:

```bash
gradle --version
```

👉 If version shows → ✅ Installation done
